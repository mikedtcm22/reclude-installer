Reclude Downloads

Choose the installer for your computer, then follow the steps below that download link.

macOS

Download Mac Installer (.pkg)

Install Steps

  1. Download the .pkg file.
  2. Open it. If macOS blocks it, Finder right-click the file and choose Open.
  3. If macOS still blocks it, go to System Settings > Privacy & Security and allow it to open.
  4. After install, fully quit and reopen Excel, then launch Reclude from Add-ins.

Windows

Download Windows Installer (.exe)

Install Steps

  1. Download the .exe file.
  2. If Windows shows a protection warning, click More info and then Run anyway.
  3. Complete the installer, then fully quit and reopen Excel.
  4. Open Add-ins and launch Reclude.
These are beta installers. If Reclude does not appear immediately after installation, fully restart Excel once and try again.
