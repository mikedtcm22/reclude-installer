Install Steps

  1. Download the .pkg installer from this page.
  2. Open the package and complete the install flow.
  3. Open Excel Desktop.
  4. Open Add-ins and launch Reclude.

Unsigned beta note: If macOS blocks opening the installer, use Finder right-click on the package and choose Open once.
