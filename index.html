Install Reclude for Excel

Private beta installers for Excel Desktop.

macOS

Download Reclude for Excel (.pkg)
  1. Download the .pkg installer.
  2. Open the package and complete the install flow.
  3. If macOS blocks opening the installer, right-click the package in Finder and choose Open once.
  4. Open Excel Desktop and launch Reclude from Add-ins.

Windows

Download Reclude for Excel (.exe)
  1. Download the .exe installer.
  2. If Windows shows a security prompt, choose More info and then Run anyway.
  3. Complete the installer flow.
  4. Open Excel Desktop and launch Reclude from Add-ins.
